Вазифа: React ва JSONPlaceholder API ёрдамида оддий вазифалар рўйхатини яратиш
Мақсад: Reactда оддий "Вазифалар рўйхати" иловасини яратиш, у JSONPlaceholder API билан алоқа қилиб, вазифаларни кўрсатиш ва бошқариш имконини беради.
Асосий талаблар:
1. Илова функционали:
   - API орқали олинган вазифалар рўйхатини кўрсатиш (JSONPlaceholder вазифалар маълумотлари намуналарини тақдим этади).
   - Рўйхатга янги вазифа қўшиш имконияти (қўшиш фақат кўриниш жиҳатидан бўлиши мумкин, серверда сақланмайди).
   - Содда ва тушунарли фойдаланувчи интерфейси.
2. API билан ишлаш:
   - Вазифалар рўйхатини олиш учун JSONPlaceholder API'дан фойдаланиш.
   - API'га сўровлар юбориш ва жавобларни қайта ишлаш.
3. React-хусусиятлари:
   - Функционал компонентлардан фойдаланиш.
   - Хуклар ёрдамида ҳолатни бошқариш (масалан, useState).
Натижалар:
- Лойиҳанинг манба коди GitHub'га юкланиши керак.
- Илованинг демонстрацион версияси қараб чиқиш учун қулай бўлган ҳар қандай хостингда мавжуд бўлиши мақсадга мувофиқ.
